## Project

This project is a simple website hosted in Firebase to support a special fundraising event. The event is a golf tournament and concert. A press release and sponsor sheet can be found in the /docs folder. Tickets to the event should be purchased at this URL:

https://www.eventbrite.com/e/sing-and-swing-tickets-1985255411377?utm-campaign=social&utm-content=attendeeshare&utm-medium=discovery&utm-term=listing&utm-source=wsa&aff=ebdsshwebmobile

## Getting Started

Read the event information in /docs. Create a single page website with information about the event, a button to buy tickets, information about donor tiers, and a contact form. The website should be golf and concert themed with colors derived from /images/logo.png. Contact form submissions should be sent to redwave@chavescounty.gop

## Architecture

Create for firebase. Format using bootstrap. Add to github.
